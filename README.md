# 🚀 Space Shooter

A modern HTML5 Canvas space shooter featuring cinematic boss battles, dynamic audio, particle effects, power-ups, adaptive gameplay mechanics, and smooth animations. Built entirely with vanilla JavaScript, HTML5 Canvas, and the Web Audio API—no external game engine required.

![Game Banner](https://img.shields.io/badge/HTML5-Canvas-orange?style=for-the-badge&logo=html5)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow?style=for-the-badge&logo=javascript)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

## 🎮 Demo

Clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/deepanshutiwari27/Spaceshooter.git
cd Spaceshooter
```

Open:

```
index.html
```

No installation required.

# ✨ Features

## 🚀 Fast-paced Arcade Gameplay

- Smooth spaceship controls
- Endless enemy waves
- Progressive difficulty
- Increasing enemy spawn rate
- Score tracking
- Lives system

---

## 👾 Boss Battles

Fight giant bosses featuring

- Multiple hit points
- Health bar
- Cinematic warning animation
- Special sound effects
- Boss destruction explosion

---

## 🎯 Aim Assist

Toggle intelligent aim assist anytime.

Features:

- Auto target nearby enemies
- Visual targeting reticle
- Toggle ON/OFF during gameplay

---

## ⚡ Rapid Fire Power-up

Collect energy powerups to unlock

- Double fire rate
- Animated timer
- HUD indicator
- Visual effects

---

## 💥 Particle Explosion System

Enemies explode into

- Fire particles
- Debris
- Sparks
- Blood-like effects

Every explosion is procedurally generated.

---

## 🌌 Dynamic Starfield

Two independent star systems

- Animated fullscreen background
- Moving in-game stars
- Infinite scrolling

Creates a parallax space effect.

---

## 🔊 Dynamic Audio

Built completely using the Web Audio API.

Includes

- Laser sounds
- Enemy explosions
- Boss explosions
- Background music
- Level-up fanfare
- Procedural sound generation

No audio assets are used.

---

## 📈 Level Progression

Difficulty increases automatically.

Each level

- Faster enemy spawning
- Level announcement
- Sound effects
- HUD updates

---

## ❤️ HUD

Real-time display of

- Score
- Lives
- Current Level
- Boss Health
- Rapid Fire Timer

---

## 🎨 Visual Effects

- Neon glow
- Animated particles
- Dynamic gradients
- Boss warning overlay
- Engine flames
- Glowing bullets
- Smooth animations

# 🎮 Controls

| Key | Action |
|------|--------|
| ← / A | Move Left |
| → / D | Move Right |
| Space | Shoot |
| Mouse Click | Shoot |
| Shift | *(Reserved for future abilities)* |
| R | Restart Game |
| Aim Assist Button | Toggle Aim Assist |

# 🎯 Why This Project?

This project was built to demonstrate what can be achieved using only native web technologies.

It showcases

- Canvas rendering
- Game loops
- Collision detection
- Procedural audio
- Particle systems
- Animation
- UI design
- Game state management

without relying on external libraries.
